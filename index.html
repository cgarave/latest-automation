Guide

An intensive guide on how to use the automation

TNC AUTOMATION - A system that automates the creation of TNC HTML file. Users can copy/paste their TNCs from docx file or create their own from scratch using the automation's power editor.

TNC - Abbreviation for Terms and Conditions.

CHANGELOG - A modal window displaying a list of recent updates, changes, and improvements made to the TNC automation tool.

TNC FILE NAME - An input field where users specify the file name for the TNC to be deployed. Format example: [Region]_SPPT_06_Jul_2025_003.

CONVERT APS - A button that generates the appropriate TNC filename by converting the entered APS code.

RESET CONTENT - A button that clears and resets the content within the text editor.

SELECT REGION - A dropdown menu listing all available Taptap regions.

SELECT TEMPLATE - A dropdown menu displaying available TNC templates. This option becomes visible only after a region is selected.

TNC TEMPLATE - A predefined and editable template specific to a selected region, which appears in the editor upon template selection.

PREVIEW - A live preview that mirrors how the TNC content will appear on the website once published.

IMPORT HTML - A feature that allows users to upload an existing TNC HTML file for editing within the tool.

DOWNLOAD - A function that converts the editor’s content into a downloadable HTML file.

EDITOR - A built-in text editor used to create, modify, or format the TNC content.
